# Insurance-Claims-Analysis
Insurance claims data analysis using Python for EDA, visualization, and hypothesis testing.

## Overview
This project performs exploratory data analysis on insurance claims data to identify patterns in customer claims, fraud cases, and claim trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Data cleaning
- Dataset merging
- Missing value handling
- Feature engineering
- Fraud analysis
- Visualization
- Hypothesis testing

## Outcome
Generated insights into customer claim behavior and fraudulent policy detection.
